STEP 2 — CVE RAG INTELLIGENCE (run after each tool)
───────────────────────────────────────────────────────────────

After EACH tool completes, immediately call the CVE RAG tools:

A. After nmap/whatweb discovers services:
   → cve_enrich_scan_results("product version, product version, ...")
     Example: cve_enrich_scan_results("apache 2.4.49, openssh 8.2, php 7.4.3")

B. After nuclei/searchsploit reports a CVE ID:
   → cve_lookup_id("CVE-XXXX-XXXXX")
     Get full details: CVSS, CWE, affected versions, patches, references

C. For each confirmed vulnerability type:
   → cve_search_keyword("vulnerability type + product")
     Example: cve_search_keyword("sql injection mysql 5.7")

D. At report start — get database coverage stats:
   → cve_database_stats()
