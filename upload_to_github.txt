# Commands to Upload to GitHub
# Repository: https://github.com/RISLAN-MOH-TM/PCP-project-.git

# Step 1: Initialize git (if not already done)
git init

# Step 2: Add remote repository
git remote add origin https://github.com/RISLAN-MOH-TM/PCP-project-.git

# Or if remote already exists, update it:
# git remote set-url origin https://github.com/RISLAN-MOH-TM/PCP-project-.git

# Step 3: Add all files
git add .

# Step 4: Check what will be committed
git status

# Step 5: Commit with message
git commit -m "feat: Kali MCP Server - Research Project

- Simplified setup for Kali VM + Windows + Claude architecture
- Default API key (kali-research-project-2024) for research use
- Clear documentation for VM-based setup
- Comprehensive troubleshooting guide
- Security hardened (10/10 score)
- Support for 14+ penetration testing tools

Architecture:
- Kali Linux VM (runs kali_server.py)
- Windows Host (runs mcp_server.py + Claude)
- Claude AI integration via MCP protocol

Documentation:
- README_FIRST.txt - Quick start overview
- HOWTO.md - Simple step-by-step guide
- PROJECT_SETUP.md - Complete architecture setup
- TROUBLESHOOTING.md - Common issues & solutions
- README.md - Project overview"

# Step 6: Push to GitHub
git push -u origin main

# If main doesn't work, try master:
# git push -u origin master

# If you need to force push (overwrites remote):
# git push -f origin main
